# Project Name: Ultimate Calculator

## Project Description

The Ultimate Calculator is a console-based Java application designed to perform basic arithmetic operations and manage stored values. The program provides users with an interactive menu system that allows them to perform calculations, store values, and analyze stored data. It also includes exception handling to ensure the program runs smoothly even when invalid input is entered.

## Project Tasks

* **Task 1: Set up the development environment**

  * Install Java Development Kit (JDK)
  * Set up IDE (VS Code)
  * Configure project files

* **Task 2: Design the application**

  * Plan menu structure and user interaction
  * Identify required features (math operations, memory, error handling)

* **Task 3: Develop the calculator functionality**

  * Implement addition, subtraction, multiplication, and division
  * Display results clearly to the user

* **Task 4: Implement exception handling**

  * Handle invalid numeric input
  * Prevent division by zero errors

* **Task 5: Implement memory features**

  * Store, retrieve, and clear a single value
  * Add and manage a list of values

* **Task 6: Develop collection features**

  * Store multiple values using ArrayList
  * Calculate sum and average of stored values

* **Task 7: Test the application**

  * Run multiple test cases
  * Debug input errors and logic issues

* **Task 8: Finalize the application**

  * Clean up code and remove unnecessary lines
  * Add comments and documentation (was honestly done as it was being created)

* **Task 9: Document the project**

  * Create README file
  * Describe features and usage

## How to Run the Program

1. Open the project in your Java IDE
2. Compile both files:

   * App.java
   * MemoryCalculator.java
3. Run the `App` class
4. Follow the menu prompts in the console

## Features

* Basic arithmetic operations (Add, Subtract, Multiply, Divide)
* Error handling for invalid input
* Protection against division by zero
* Single value memory storage
* List-based memory storage (up to 10 values)
* Sum and average calculations for stored values
* User-friendly menu system
